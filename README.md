# 🫀 Heart Disease Prediction AI Web App

This project uses a PyTorch neural network trained on clinical health data to predict cardiovascular disease risk.

## 🚀 Features
- Neural network trained on medical dataset
- Flask-based web interface
- User-friendly clinical input form
- Real-time AI prediction (risk / no risk)
- Professional medical UI dashboard

## 🧠 Tech Stack
- Python (PyTorch)
- Flask
- HTML/CSS
- Machine Learning

## 📊 Model Input Features
- Age, Sex
- Chest pain type
- Blood pressure
- Cholesterol
- ECG results
- Heart rate
- Exercise indicators

## ⚠️ Disclaimer
This tool is for educational purposes only and is not a medical diagnosis system.
